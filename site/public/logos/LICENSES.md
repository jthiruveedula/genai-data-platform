# Logo assets

The SVGs in this folder are **placeholder monograms**, not the official AWS
smile, Microsoft Azure, or Google Cloud marks. They exist so theming and
layout can be built and reviewed now.

Before this site is presented as production-ready (per `PLAN.md` §7.3), swap
each placeholder for the official mark under nominative use, downloaded from:

| Flavor | Official source | Brand guideline |
|---|---|---|
| AWS | https://aws.amazon.com/architecture/icons/ | https://aws.amazon.com/trademark-guidelines/ |
| Azure | https://learn.microsoft.com/en-us/azure/architecture/icons/ | https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks |
| GCP | https://cloud.google.com/icons | https://cloud.google.com/brand-resource-center |
| OSS | Kubernetes, PostgreSQL, Qdrant, etc. — each project's own brand kit | per-project |

Rules: never recolor, distort, or imply endorsement. Each usage carries a
"Not affiliated with…" note (see the site footer).
